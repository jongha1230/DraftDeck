# MVP Scope

## 포함 범위

- source import
- source 기반 draft generation
- selection-based AI editing
- autosave
- revision history
- Markdown export

## 제외 범위

- real-time collaboration
- multi-device sync
- analytics dashboard
- PDF export
- public multi-user publishing workflow
- AI fact-check pipeline
- prompt injection 완전 차단

## 왜 제외했나

- DraftDeck의 포트폴리오 강점은 "AI API를 붙였다"가 아니라 writing workflow 경계를 분명히 보여주는 것이다.
- 범위를 넓히기보다 source -> AI assist -> autosave -> revision 기록 흐름을 신뢰 가능하게 만드는 것이 우선이다.
